# K线案例训练系统

> 老冯交易系统 - 每日20图训练计划

## 📚 简介

这是一个用于训练K线形态识别的系统，帮助交易者每天收集和分析K线案例，建立形态记忆。

### 核心功能

- ✅ 每日进度追踪 (0/20)
- 🏷️ 形态分类统计 (双底、左峰突破、阳阴阳)
- 📸 案例收集与管理
- 💾 本地数据存储
- 🔍 按形态筛选查看
- 🤖 AI辅助识别股票名称

## 🎯 使用方法

**在线访问**: https://lvtizan.github.io/kline-trainer/kline-cases.html

**仓库地址**: https://github.com/lvtizan/kline-trainer

或本地打开:
   ```
   kline-cases.html
   ```

2. **添加案例**
   - 点击"➕ 添加新案例"
   - 上传K线截图
   - 选择形态分类
   - 填写特征描述
   - 提交

3. **查看进度**
   - 每日目标：20个案例
   - 进度条自动更新
   - 完成后显示庆祝提示

4. **股票识别**
   - 上传图片后说"识别图片"
   - AI自动识别并填入股票代码
